# snake-game-JavaScript

## Click Here to Play the Live Game! (https://ahmedawan216.github.io/snake-game-JavaScript/)

A responsive, logic-based Snake Game built with Vanilla JavaScript. This project demonstrates core programming concepts like game loops, coordinate systems, and dynamic DOM manipulation.

## Key Features
* Logic-Driven: Developed using pure JavaScript (ES6+) without external libraries.
* Interactive UI: Features smooth control response and real-time score tracking.
* Responsive Design: Playable across different screen sizes and devices.
* Clean Code: Structured for high performance and easy readability.

## Tech Stack
* HTML5: Game structure and canvas rendering.
* CSS3: Custom styling and responsive layout.
* JavaScript: Game engine logic, event handling, and state management.

## How to Play
1. Use the arrow keys to move the snake.
2. Eat the food to grow and increase your score.
3. Game over occurs if you hit the walls or yourself.
